### Module 1: Python Programming for Data Science
- [ ] Introduction to Python
- [ ] Variables, Data Types, Operators & Expressions
- [ ] Conditional Statements, Loops & Iterations
- [ ] Functions & Modules
- [ ] File Handling & Exception Handling
- [ ] OOP Basics
- [ ] NumPy Fundamentals
- [ ] Pandas Fundamentals
- **Mini Projects:** Student Management System · Sales Data Analysis · Data Cleaning Exercises

### Module 2: Data Analysis & Visualization
- [ ] Data Collection & Preprocessing
- [ ] Missing Value Treatment & Outlier Detection
- [ ] Exploratory Data Analysis (EDA)
- [ ] Data Transformation & Feature Engineering Basics
- [ ] Visualization: Matplotlib, Seaborn, Plotly
- [ ] Charts: Line, Bar, Histogram, Scatter, Box Plot, Heatmap, Correlation Matrix
- **Mini Project:** E-Commerce Sales Dashboard

### Module 3: Statistics & Mathematics for ML
- [ ] Descriptive Statistics & Probability Theory
- [ ] Normal Distribution & Hypothesis Testing
- [ ] Correlation & Covariance
- [ ] Linear Algebra: Vectors & Matrices
- [ ] Gradient Descent Concept

### Module 4: Machine Learning Fundamentals
- [ ] Regression vs Classification, Train/Test Split, Model Evaluation
- [ ] Regression: Linear, Multiple Linear, Polynomial
- [ ] Classification: Logistic Regression, Decision Tree, Random Forest, KNN, SVM, Naive Bayes
- [ ] Clustering: K-Means, Hierarchical
- [ ] Evaluation Metrics: Accuracy, Precision, Recall, F1, ROC-AUC
- **Mini Projects:** House Price Prediction · Employee Salary Prediction · Customer Segmentation · Loan Approval Prediction

### Module 5: Deep Learning (TensorFlow & Keras)
- [ ] ANN: Perceptron, Activation Functions, Feed Forward Networks
- [ ] TensorFlow & Keras: Building, Compiling, Training Models
- [ ] Backpropagation, Optimizers, Loss Functions, Hyperparameter Tuning
- [ ] CNN: Image Processing, Feature Extraction, Image Classification
- [ ] RNN & LSTM: Sequence Prediction, Time Series Forecasting
- **Projects:** Handwritten Digit Recognition · Face Mask Detection · Image Classification System · Stock Price Prediction

### Module 6: Natural Language Processing (NLP)
- [ ] Text Processing: Tokenization, Stop Word Removal, Lemmatization, Stemming
- [ ] Libraries: NLTK, SpaCy, TextBlob
- [ ] Feature Extraction: Bag of Words, TF-IDF, Word Embeddings
- [ ] Applications: Sentiment Analysis, Text Classification, Spam Detection, Chatbot Development
- [ ] Intro to Transformers: BERT, GPT Models, Hugging Face Basics
- **Projects:** Movie Review Sentiment Analysis · Spam Email Detection · Resume Screening System · AI Chatbot

### Module 7: Capstone Project & Deployment
- [ ] Project Lifecycle: Problem Identification → Dataset Collection → Data Cleaning → Model Building → Evaluation → Deployment
- [ ] Deployment Tools: Streamlit, Flask, GitHub
- [ ] Dashboard Integration: Interactive AI Apps, Visualization Reports
